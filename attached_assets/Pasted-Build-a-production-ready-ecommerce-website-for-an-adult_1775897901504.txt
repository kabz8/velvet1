Build a production-ready ecommerce website for an adult wellness / intimate products store in Kenya. The store must feel premium, discreet, tasteful, moodful, modern, and conversion-focused — never vulgar or overly explicit. The overall visual style should be elegant, soft, dark-luxury, and trustworthy, with a private-shopping feel.

Core stack:
- Next.js 15 + TypeScript + App Router
- Tailwind CSS
- shadcn/ui
- Supabase for:
  - PostgreSQL database
  - Authentication
  - Storage for media uploads
- Prisma ORM or Supabase-native queries
- Ready for deployment on Vercel
- Optimized for Replit development
- Fully responsive
- SEO optimized

Important business rules:
- Currency: KES only
- Nairobi County delivery fee: KES 300
- Outside Nairobi delivery fee: KES 450
- Guest checkout allowed
- Anonymous checkout optional
- Phone number is always required for delivery arrangements
- All product images, banner images, and files must be uploaded directly from the dashboard using file upload inputs
- Do NOT require manual backend file URLs for images/files
- Site must feel discreet, luxurious, trustworthy, and private

Brand direction:
- Premium, elegant, private, romantic, tasteful
- Dark theme by default

- Classy and discreet language throughout

Add an 18+ entry gate:
- On first visit, show a beautiful full-screen age confirmation modal/page
- Messaging should be tasteful and discreet, e.g. “This store contains products intended for adults only”
- User must confirm they are 18+ to enter
- Include “I am 18+” and “Exit” actions
- Save confirmation in a cookie or local storage so the gate does not appear on every page load
- Gate should be manageable from admin settings:
  - enable/disable
  - edit title
  - edit message
  - edit button labels
  - background image/banner upload
- The age gate should match the premium moodful design of the site
